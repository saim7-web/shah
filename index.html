Projects

Robotic Hand

Developed a sophisticated servo-powered robotic hand capable of replicating intricate human hand movements. This project involved precise sensor integration to mimic grasping and manipulation, combined with advanced OpenCV techniques for object recognition and interaction, demonstrating fine motor control and intelligent perception.

Tech: Python, Arduino, OpenCV, Servo Control, I2C Communication

Mars Rover

Engineered an autonomous 6-wheel Mars Rover designed for exploration and obstacle avoidance in challenging terrains. Equipped with an array of IR sensors and programmed with Python, the rover features intelligent navigation algorithms allowing it to autonomously detect and maneuver around obstacles, simulating off-world exploration capabilities.

Tech: Raspberry Pi, IR Sensors, Python, Autonomous Navigation, RTOS

Automated Conveyor Belt System

Designed and implemented an automated conveyor belt system for efficient object sorting. This system utilizes a combination of IR and color sensors to identify and categorize objects based on predefined criteria, automating the material handling process with precision and speed.

Tech: Arduino, IR Sensors, Color Sensors, Servo Motors, PLC Logic, Industrial Automation

Autonomous Car

Constructed a self-driving model car featuring real-time object and lane detection capabilities. Utilizing Python and OpenCV, the car processes live video feeds to identify objects and maintain lane adherence, while ultrasonic sensors provide critical proximity data for collision avoidance, showcasing advanced perception and control systems.

Tech: Python, OpenCV, Ultrasonic Sensors, Real-time Object Detection, Kalman Filters

Hexapod Spider Robot

Built a complex 6-legged hexapod spider robot with 12 independently controlled servos, enabling highly realistic and dynamic walking and turning gaits. Programmed in C++ with an Arduino and PWM driver, this project demonstrates intricate kinematics and synchronized motion control for multi-legged locomotion.

Tech: Arduino, PWM Driver, C++, Inverse Kinematics, Servo Feedback
